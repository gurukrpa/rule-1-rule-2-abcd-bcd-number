## 🚀 **Quick Start Commands:**

### Before Starting New Logic:
```bash
git checkout -b step-N-description
```

### Save Progress:
```bash
git add . && git commit -m "Your progress message"
```

### If Logic Works:
```bash
git checkout main
git merge your-branch-name
git branch -d your-branch-name
```

### If Logic Breaks:
```bash
git checkout main
git branch -D your-branch-name
```
